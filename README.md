# AULA_POO-BCOPSTEIN_03-11-vaadin

# Enunciado
1) Crie um formulário para solicitar dados sobre uma pessoa (nome, peso, altura (em cm)). O peso ideal da pessoa é calculado pela fórmula: * Peso normal = altura - 100 *

A partir do cálculo do peso ideal, exibir notificações sobre a saúde da pessoa conforme o que segue:
- Até 10% abaixo do ideal: Ótimo 
- Mais de 10% abaixo do ideal: abaixo do peso 
- Até 10% acima do ideal: Bom 
- Entre 10% e 30% acima do ideal: Atenção 
- Mais de 30% acima do ideal: Sobre Peso

2) Crie uma classificação para a altura da pessoa e inclua a classificação da altura juntamente com a da altura.

3) Crie um "grid" para manter na tela os dados de todas as pessoas que fizerem consultas.

4) Verifique novas estatísticas de saude que possam ser calculadas (IMC, etc). Altere a classe "Pessoa" se for necessário
